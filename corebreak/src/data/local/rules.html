RULES

Corebreak is a mathematical coin game, clone of Spherebreak mini-game from Final Fanstasy X-2, where players must create specific number combinations within a time limit.

Basic Rules

The game board consists of a center coin (Core Sphere) surrounded by multiple coins (Entry Spheres). Players must select coins that multiply together to create numbers ending in the target number (Multiple). At least 3 coins must be used in each combination. The game proceeds in turns (phases), with each successful combination earning points.

Scoring System

Base points are awarded for each valid combination Bonus points are earned for:

  • Using more coins in a combination
  • Creating consecutive successful combinations (Chain Bonus)
  • Completing combinations quickly
  • Meeting special conditions like using specific multiples

Advanced Mechanics

Echo Effect: Reusing numbers from previous combinations for bonus points Fever Mode: Activated after multiple successful combinations, increasing point multipliers Critical Break: Achieved by meeting specific numerical conditions, awarding substantial bonus points
